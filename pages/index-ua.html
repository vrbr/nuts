Вимоги до макетів

  • Цифровий друк

    Для цифрового друку

    • Формат файлів: векторні (CDR, PDF), растрові (TIFF, JPEG)
    • Колірна модель: CMYK
    • Розмір файлу – не більше 50 Мб
    • Розмір файлу повинен відповідати друкованому відбитку (1:1)
    • Для JPEG-файлів не використовувати стиснення
    • Для TIFF-файлів - звести всі шари в один і при збереженні вибирати ZIP або LZW стиснення
    • Для PDF-файлів - всі шрифти мають бути впроваджені у PDF
    • Роздільна здатність: 300 dpi
    • Макет не повинен містити міток обрізу, штампів, біговок та будь-яких інших об'єктів, що не друкуються.

  • Широкоформатний друк

    Для широкоформатного друку

    • Формат файлів: TIFF, EPS
    • Растрова графіка в натуральному розмірі (1:1)
    • Роздільна здатність: 100 dpi
    • Колірна модель: CMYK
    • Чорний колір має бути композитним: C-60% M-60% Y-60% K-100%
